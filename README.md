# Voice Math Game (ボイス計算ゲーム)

声で答える、シンプルで直感的な計算トレーニングゲームです。キーボードやマウスを使わずに、マイクに向かって数字を言うだけで足し算・引き算の計算に挑戦できます。

## 🚀 特徴
* **完全ハンズフリー操作**: Web Speech API を使用し、音声のみで回答が可能です。
* **シンプルなルール**: 90秒間で何問解けるか、シンプルな算数問題（足し算・引き算）が出題されます。
* **フィードバック機能**: 
    * 正解時の視覚的なエフェクト（赤い二重丸）。
    * Web Audio API によるビープ音。
    * 音声合成（Speech Synthesis）による「スタート」「終了」の読み上げ。
* **マルチデバイス対応**: PC、タブレット、スマートフォンなど、各画面サイズに最適化されたレスポンシブデザイン。

## 🛠 使用技術
* **HTML5 / CSS3**: ダークモードを基調とした視認性の高いUI。
* **JavaScript (Vanilla JS)**: 外部ライブラリを使用しない軽量な実装。
* **Web Speech API**: 音声認識（`SpeechRecognition`）および音声合成（`SpeechSynthesis`）。
* **Web Audio API**: ゲーム内サウンド生成。

## 🎮 遊び方
1. ブラウザでファイルを開きます。
2. 画面をクリック、または `Enter` キーを押してゲームを開始します。
3. マイクの使用許可を求められたら「許可」を選択してください。
4. 画面中央に表示される計算式の答えを、マイクに向かって声で答えます（例：「3 + 5」なら「はち」）。
5. 制限時間は90秒です。ハイスコアを目指しましょう！

### ショートカットキー
* `Enter`: ゲームの開始 / リスタート
* `F`: 全画面表示の切り替え

## 📦 デプロイ方法 (GitHub Pages)
このリポジトリを GitHub Pages で公開することで、URLにアクセスするだけですぐに遊ぶことができます。

1. GitHubのリポジトリ設定（Settings）を開きます。
2. **Pages** セクションを選択します。
3. Build and deployment の Source で **Deploy from a branch** を選択し、`main` ブランチを保存します。
4. 発行されたURLにアクセスします。
    * ※音声認識機能を使用するため、`https` 環境（GitHub Pages等）での実行を推奨します。
